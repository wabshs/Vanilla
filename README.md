# 项目介绍
>本项目是基于SpringBoot和Vue的智慧校园系统，用户端采用小程序的形式，管理员端采用Web的形式

# 后端技术栈
1. 核心框架：SpringBoot
2. 安全框架：SpringSecurity
3. ORM框架：Mybatis、Mybatis-Plus、~~Mybatis-Flex~~
4. NoSQL：Redis
5. Markdown 转 HTML：commonmark-java

# 前端技术栈
1. 核心框架：Vue2.x Vue-Router
2. UI框架：Element-UI

# 前端运行
1. cd ./vanilla_front
2. npm install
3. npm run serve

# 后端运行
1. 在本机的数据库运行SQL文件
2. 打开Redis
3. 装好依赖直接运行主类即可
